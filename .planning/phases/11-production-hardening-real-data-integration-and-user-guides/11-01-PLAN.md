# Phase 11, Plan 01: Supabase RLS Hardening (SQL Migration)

## Goal
Implement a consolidated SQL migration to ensure all tables (Core, Wellness, Couples, Family, etc.) have hardened Row Level Security (RLS) policies. This ensures that when we drop mock data, the real data is properly isolated to users and circles.
